# Plan Fase 2 - Admin

## Objetivo
Habilitar backoffice robusto para gestionar el catalogo tecnico.

## Alcance
- Filament panel en `/admin`.
- CRUD de `Systems`, `Modules`, `Endpoints`, `Artefacts`.
- CRUD de `Users`, `Roles`, `Permissions`.
- Publicacion de endpoints restringida por permiso.

## Criterios de salida
- Editor/Admin pueden mantener catalogo completo.
- Viewer queda fuera del panel.
- Reglas RBAC aplicadas en recursos admin.
